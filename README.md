# Ugly Sweater

## Get Started

```sh
yarn install
yarn chain # starts hardhat chain
yarn start # starts webapp
# yarn generate: creates deployer for builder deploys
yarn deploy # deploys latest contract
# yarn surge: deploys to public domain
```
